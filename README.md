# MERN Bug Tracker

A comprehensive bug tracking application built with the MERN stack, featuring robust testing and debugging practices.

## Features

- Report, view, update, and track bugs
- Filter bugs by status and priority
- Detailed bug information with environment details
- Comprehensive testing suite
- Error handling and debugging tools

## Installation

### Prerequisites
- Node.js (v14 or higher)
- MongoDB
- npm or yarn

### Backend Setup
```bash
cd server
npm install
cp .env.example .env
# Configure your .env file with MongoDB URI and other settings
npm run dev
